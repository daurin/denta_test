# denty_cloud_test

He utilizado arquitectura limpia, la misma arquitectura que estoy utilizando ultimamente para mis proyectos personales.

## Dependencias

- **auto_route**: Navegación y rutas automáticas tipo declarativo.
- **get_it**: Inyección de dependencias y localizador de servicios.
- **http**: Realizar peticiones HTTP a APIs REST.
- **get_storage**: Almacenamiento local rápido y sencillo.
- **path_provider**: Acceso a rutas del sistema de archivos (documentos, caché, etc), get_storage lo requiere.
